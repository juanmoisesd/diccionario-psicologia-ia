# Cómo citar

Esta página proporciona los formatos de citación listos para copiar en los estilos más comunes.

**DOI permanente**: `10.5281/zenodo.19297225`

---

## APA 7

```
de la Serna, J. M. (2026). Diccionario de Psicología en Español para IA (v1.0.0) [Dataset]. Zenodo. https://doi.org/10.5281/zenodo.19297225
```

---

## BibTeX

```bibtex
@dataset{delaserna_diccionario_psicologia_2026,
  author    = {de la Serna, Juan Mois{\é}s},
  title     = {Diccionario de Psicolog{\í}a en Espa{\ñ}ol para IA},
  year      = {2026},
  version   = {1.0.0},
  publisher = {Zenodo},
  doi       = {10.5281/zenodo.19297225},
  url       = {https://doi.org/10.5281/zenodo.19297225},
  license   = {CC BY 4.0}
}
```

---

## IEEE

```
J. M. de la Serna, "Diccionario de Psicología en Español para IA," version 1.0.0, Zenodo, Mar. 2026. [Dataset]. doi: 10.5281/zenodo.19297225.
```
